## Installation
<a name="Installation"/>

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the flask.


```bash
$ mkdir myproject
$ cd myproject
$ python3 -m venv venv
. venv/bin/activate
pip install Flask
```
